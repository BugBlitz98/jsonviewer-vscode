# JSON Viewer Tools: Interactive Graph Visualizer for VS Code

Transform your complex JSON files into beautiful, **interactive graphs** directly inside Visual Studio Code! 
Are you tired of scrolling through thousands of lines of nested JSON arrays and objects? **JSON Viewer Tools** is the ultimate VS Code extension designed specifically for developers, data analysts, and API engineers who need an intuitive, visual approach to explore and comprehend intricate JSON data structures. By securely parsing your local JSON files and converting text-based JSON into a dynamic node-based visualization, this extension accelerates debugging, API response inspection, and data analysis algorithms.

## ✨ Key Features
- 📂 **Instant Open JSON Visualizer** – Quickly render your JSON files with a single command (`jsonVisualizer.openPanel`) or via the editor context menu.
- 📊 **Interactive Graph View** – Easily navigate through deeply nested JSON structures and arrays using a modern, visual diagram interface.
- 🔍 **Advanced Zoom & Pan Navigation** – Built-in controls to pan across sprawling datasets and zoom in/out with precision for better spatial data exploration.
- 🎯 **Privacy First & Fully Local** – Everything is processed securely on your local machine within the VS Code webview. None of your JSON data is sent to external servers.
- 🚀 **Boost Productivity & Debugging** – Perfect for inspecting REST API responses, configuration metrics, Webpack/NPM files, and debugging complex JSON payloads.

---

## 📦 Installation
1. Open Visual Studio Code.
2. Go to the **Extensions** view by clicking on the Extensions icon in the Activity Bar or pressing `Ctrl+Shift+X` (`Cmd+Shift+X` on Mac).
3. Search for **"JSON Viewer Tools"** or **"JSON Viewer: Interactive Graph Visualizer"**.
4. Click **Install**.

---

## 🚀 Usage guide
1. Open any `.json` file in your VS Code editor.
2. Run the visualizer using one of the following methods:
   - **Right-Click Menu:** Right-click anywhere in the JSON file and select **Open JSON Visualizer**.
   - **Command Palette:** Press `Ctrl+Shift+P` (or `Cmd+Shift+P` on Mac), search for and execute **Open JSON Visualizer**.
3. A split-panel webview will open displaying your JSON format properly decoded as an interactive graph!

---

## 🔧 System Requirements
- Visual Studio Code v1.103.0 or higher
- A valid, syntax-error-free `.json` file

## 🛠 Commands

| Command                   | Description                                                                 |
